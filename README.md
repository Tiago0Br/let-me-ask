# Let me ask

Aplicação fullstack desenvolvida durante o evento **NLW Agents** que permite criar salas para lives onde o público pode fazer perguntas respondidas por inteligência artificial baseada na transcrição em tempo real.

## 🚀 Funcionalidades

- Criação de salas para transmissões ao vivo
- Sistema de perguntas do público
- Respostas automáticas via IA baseadas no conteúdo da live
- Transcrição em tempo real das transmissões

## 📁 Estrutura do Projeto

Este é um monorepo contendo:

- **Frontend** (`/web`) - Interface do usuário
- **Backend** (`/server`) - API e processamento de IA
